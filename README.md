# Bucheen UI

A modern responsive dating app landing page built with React and Tailwind CSS.

## Overview

Bucheen UI is a clean and modern landing page project for a dating app concept.  
The project is designed with a soft visual style, responsive layout, and reusable React components.

## Sections

- Navbar
- Hero
- Features
- Footer

## Features

- Responsive layout
- Modern UI design
- Built with React functional components
- Styled with Tailwind CSS
- Reusable and clean component structure

## Tech Stack

- React
- Tailwind CSS
- JavaScript
- Vite
